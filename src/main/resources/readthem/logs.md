# 这是什么文件夹？

> [!NOTE]
> 本目录由 System Storage Lib 创建，用于存放日志文件。

### 包含的文件

- `latest.log` — 当前会话的日志
- `1.log`, `2.log`, ... `N.log` — 已轮转的归档日志

# What is this folder?

> [!NOTE]
> This directory was created by System Storage Lib for storing log files.

### Files Included

- `latest.log` — Logs from the current session
- `1.log`, `2.log`, ... `N.log` — Rotated archive logs
