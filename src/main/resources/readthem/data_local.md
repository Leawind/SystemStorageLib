# 这是什么文件夹？

> [!NOTE]
> 本目录由 System Storage Lib 创建，用于存放持久化但本机独有的数据，或可再生但较耗时的数据。

### 特性

- 持久化：应保留，但非必要不同步
- 本机特定：可能包含依赖于硬件或操作系统的数据

# What is this directory?

> [!NOTE]
> This directory is created by System Storage Lib, and is used to store persistent but machine-local data, or regenerable but time-consuming data.

### Characteristics

- Persistent: should be retained but not necessarily synced
- Machine-specific: may contain hardware or OS-dependent data
