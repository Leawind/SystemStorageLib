# What is this directory?

> [!NOTE]
> This directory is created by System Storage Lib, and is used to store configuration files.

### Characteristics

- Persistent: user configuration should be retained long-term
- Editable: files stored in plain text formats, safe for manual editing
- Sync-friendly: recommended for version control or sync tools
